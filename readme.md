## 🛠 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/shreyu1701/codeedoc.git
   cd codeedoc
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run the project**
   ```bash
   npm start
   ```

## 🎯 Usage

1. Open `http://localhost:3000/` in your browser.
2. Enjoy Website !
